添加指定文件到缓存
git add hello.php



添加所有文件到缓存
git add .



取消已缓存的文件
git reset HEAD -- hello.php 



查看文件状态
git status -s


查看文件改动的详细信息
尚未缓存的改动：git diff
查看已缓存的改动： git diff --cached
查看已缓存的与未缓存的所有改动：git diff HEAD
显示摘要而非整个 diff：git diff --stat



将缓存区内容添加到仓库中
git commit


提交时加上备注
git commit -m '修改'


添加/备注/并提交

回退到当前版本

回退到上个版本
